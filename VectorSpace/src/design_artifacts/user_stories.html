Agnes

Scenario 1 Stories


As a compliance analyst, I want to create a task list with deadlines for each project phase, so I can ensure timely completion and meet all compliance requirements.

As a compliance analyst, I want to be able to add an estimated completion time to tasks, so I can plan out enough time for each task.

As a busy professional, I want to receive reminders a set number of days before each deadline, so I can stay on track and avoid last-minute, rushed work.

As a user, I want to automatically prioritize my tasks by urgency, so I can plan to complete urgent tasks first.

As a compliance analyst, I want to organize tasks into project phases to easily track which tasks belong to which compliance stage.

As a compliance analyst, I want a visual timeline of my project tasks, helping me to view my progress and adjust my schedule accordingly.

As a busy professional, I want an overview of upcoming deadlines, so I’m prepared for upcoming work and not overloading myself with rushing to get things done before a pressing deadline.

As a compliance analyst, I want the scheduling module to suggest efficient task order, allowing me to balance workload across all deadlines and priorities.

As a working user, I want the flexibility to adjust deadlines or add new tasks as projects change, ensuring my schedule remains current.

Scenario 2 Stories


As a compliance analyst, I want to receive announcements sorted by urgency, so I can prioritize regulatory updates quickly.

As a user, I want to have a category attached to announcements, so I know what the announcement relates to.

As a team member, I want to mark announcements as read, so I can track the announcements I am aware of.

As a team member, I want my manager to be able to view the "read" markings of an announcement by the recipient, so they know that the announcement has been read and don't need to follow up.

As a team member, I want to mark announcements as addressed, so I can archive announcements that are addressed and no longer necessary to display.

As a team member, I want my manager to be able to view the "addressed" markings of an announcement by the recipient, so they know that the contents of the announcement has been addressed and mark off the task.

As a compliance analyst, I want an announcement category for "urgent regulatory changes" to ensure high-priority information regarding compliance stands out and can be addressed quickly.

As a user and analyst, I want to search for past announcements, enabling easy reference to previous updates if I need a refresher.

As a compliance analyst, I want to archive old announcements, ensuring that my dashboard stays organized and focused on current updates.

Scenario 3 Stories


As a user, I want to be able to have custom tools, so VectorSpace is integrated into all aspects of my work and I can have most if not all tools all in one app.

As a compliance analyst, I want an automated tracking system that monitors regulatory updates, so I can stay up-to-date on industry standards.

As a compliance analyst, I want a dedicated dashboard to organize regulatory updates, so it is easier to track all compliance requirements.

As a user, I want flagged updates that directly affect my projects to stand out, so I can prioritize important changes.

As a compliance analyst, I want the tracker to filter updates by relevance, so I only see changes that apply to my work.

As a compliance analyst, I want a visual timeline of my project tasks, helping me to view my progress and adjust my schedule accordingly.

As a compliance analyst, I want the dashboard to display summaries of recent regulatory changes, so I can grasp new requirements quickly and effectively.

As a compliance analyst, I want the system tracker to log all tracked updates, so I can easily reference past changes for auditing purposes.

As a compliance professional, I want update notifications to include links to full regulatory documents, so I have quick access to complete information.

Lawrence

Scenario 1 Stories


As a graduate student and software engineer, I want to have optimized scheduling to efficiently allocate time for studying, work projects, and thesis preparation within my busy schedule.

As a working graduate student, I want to block out daily study periods, so I consistently dedicate time to my thesis and avoid last-minute preparation.

As a student with multiple deadlines, I want to be able to automatically organize my tasks by urgency and available time slots, ensuring my high-priority tasks are addressed first.

As a user, I want to receive daily reminders about my planned schedule, so I can stay focused and prepared for the day ahead.

As a user, I want to flag tasks as high-priority so they are allocated time first in my schedule, ensuring that my essential tasks are completed first.

As a student and software engineer, I want to be notified if I underestimated task completion time, so I can adjust my schedule accordingly and avoid rushing.

As a busy user, I want an overview of my schedule progress, so I can visualize my workload and better plan my time during hectic weeks.

As a student, I want to have custom notifications for critical tasks, so I don’t overlook important deadlines.

Scenario 2 Stories


As a team member, I want to receive and send urgent messages even with focus mode on for the recipient because otherwise I or the recipient may miss urgent messages.

As an office worker and team member, I want to activate focus mode to mute non-essential notifications and reduce distractions while working on projects.

As a team member, I want a focus mode status shared across my communication tools, so colleagues know not to interrupt my focus periods.

As a team member, I want to filter notifications in focus mode to receive only those from selected team members or projects, allowing me to stay on task with minimal distraction.

As a busy professional, I want to set weekly focus goals to stay disciplined and maximize productivity without distractions.

As a team member, I want my status message in focus mode to indicate what project I’m focusing on, keeping colleagues informed of what I am working on and my availability.

As a developer and student, I want reminders to activate focus mode during intense work periods, ensuring I stay undistracted during high-priority tasks.

As a user and team member, I want to use task blocks in focus mode, so I can work uninterrupted while still taking scheduled breaks to avoid burnout.

Scenario 3 Stories


As a graduate student, I want to upload thesis sources, so I can access relevant materials and notes when I need them.

As a graduate student, I want to be able to optionally add summaries to thesis sources, so I don't need to reread any texts.

As a graduate student, I want to tag and categorize sources by topic, allowing quick access to references specific to different sections of my thesis.

As a graduate student, I want to search for keywords across my sources, so I can locate key information without scrolling through every document.

As a student, I want VectorSpace to integrate with my note-taking app, allowing me to connect notes to sources, so I can view my notes with my sources.

As a graduate student, I want to have a dedicated thesis dashboard showing my writing progress, key sources, and deadlines, so I can stay organized throughout the writing process.

As a student, I want to save annotations directly in my source files, so I can keep track of important insights and arguments for my thesis.

As a graduate student, I want a summary of all my notes and tags across sources, providing me with a quick overview of my research to prepare for my thesis defense.

As a graduate student, I want to customize a module for thesis sources and related material because it would allow me to access all my sources and notes on them in a single place.

Nathaniel

Scenario 1 Stories


As a user, I want to log productivity sessions and track trend so that I can see when I am the most productive.

As a student, I want to log my study sessions with specific variables, so that I can identify patterns in my productivity.

As a student, I want to see visual representations of my productive and unproductive periods so that I can easily analyze my focus trends.

As a student, I want to compare my productivity with different variables such as when studying alone/with friends, in the morning/in the afternoon, etc., so that I can determine which environment works best for me.

As a student, I want to track how different activities, like listening to music, impact my productivity so that I can optimize my study sessions.

As a student, I want to receive reminders to study during my most productive times so that I can take advantage of my peak focus periods.

As a student, I want to create a weekly study schedule based on my productivity trends so that I can improve my study efficiency.

As a student, I want to share my productivity data with friends so that we can create study groups during our most productive times together.

As a student, I want to set personal goals for productivity, like studying for a certain number of hours each week, so that I can stay accountable.

As a student, I want to access my productivity logs easily, so I can review and adjust my study habits as needed.

Scenario 2 Stories


As a team member, I want to set goals and tasks for group projects, so that the team is aware of the list of tasks for the project.

As a team member, I want to set individual goals linked to a group project's objectives so that all group members are aware of their responsibilities.

As a team member, I want to view a visual breakdown of each team member’s progress so that I can easily see tasks' completion and what still needs to be done.

As a team member, I want to receive notifications for upcoming deadlines so that I can stay on track with my contributions.

As a team member, I want to provide feedback on each other’s tasks within the platform so that we can improve each other's work together.

As a team member, I want to see a history of completed tasks so that we can reflect on our progress.

As a team member, I want to categorize tasks based on their urgency so that we can prioritize our work effectively.

As a team member, I want to visualize the overall project timeline, so we can view our progress towards the final deadline.

Scenario 3 Stories


As a student, I want to merge my emails and announcements from different platforms into one, so I can access all updates in one place.

As a student, I want to categorize notifications by importance, so I only receive alerts for essential updates that require immediate attention and not worry about non-urgent messages.

As a student, I want to filter out non-urgent notifications so that I can focus on my studies without constant distractions.

As a student, I want to set custom notifications for different categories ( assignments, project updates, etc.), so my notifications are stay organized.

As a student, I want to review filtered notifications later at my convenience so that I don’t miss any relevant information.

As a student, I want to receive reminders for important announcements, such as class changes or assignment deadlines, so I can stay informed.

As a student, I want to mark certain notifications as important so that I can quickly refer back to them when needed.

As a student, I want to customize how I receive notifications (email, push notifications, etc.), so I can manage alerts in a way that works best for me.

As a student, I want to access a summary of missed notifications so that I can catch up without feeling overwhelmed.
