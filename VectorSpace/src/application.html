Running VectorSpace prototype

Running Protype (Building Image and Running in Container)


Clone the repository: VectorSpace on Github
From within the "vectorspaceapp" directory proceed to the following docker sections to either run the application prototype or unit tests.

Running Protype (Building Image and Running in Container)


Within "vectorspaceapp" directory.

$ docker build -f Dockerfile -t vectorspaceapp:latest .

$ docker run -dp 8000:3000 --name react-container vectorspaceapp:latest

Open localhost:8000 in browser.

Running Protype (Building Image and Running in Container)


Within "vectorspaceapp" directory.

$ docker build -f Dockerfile.test -t vectorspaceapp:tests .

$ docker run -dp 8000:3000 --name react-container vectorspaceapp:latest

Open localhost:8000 in browser.
